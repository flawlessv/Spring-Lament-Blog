# 启动指南

## 启动命令

```bash
# 1. 启动 ChromaDB
npx chromadb run --path ./data/chroma --port 8000

# 2. 启动 Ollama（新开终端）
ollama serve

# 3. 启动项目（再开一个终端）
npm run dev
```

## 首次启动

```bash
# 1. 安装 Ollama（macOS）
brew install ollama

# 2. 拉取模型
ollama pull nomic-embed-text

# 3. 启动 ChromaDB
npx chromadb run --path ./data/chroma --port 8000

# 4. 启动 Ollama（新开终端）
ollama serve

# 5. 启动项目（再开一个终端）
npm run dev
```

## 检查服务

```bash
# 检查 ChromaDB
curl http://localhost:8000/api/v1/heartbeat

# 检查 Ollama
curl http://localhost:11434/api/tags
```
