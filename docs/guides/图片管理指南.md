## 🖼️ 如何使用本地图片

### 1. 文章封面图

在创建或编辑文章时，使用本地路径：

```
/images/posts/covers/tech-1.jpg
```

或者在代码中直接引用：

```tsx
<Image
  src="/images/posts/covers/tech-1.jpg"
  alt="文章封面"
  width={1200}
  height={630}
/>
```

### 2. 文章内容配图

在 Markdown 中使用：

```markdown
![图片说明](/images/posts/content/example-1.png)
```

或使用 HTML：

```html
<img src="/images/posts/content/example-1.png" alt="图片说明" />
```

### 3. 个人爱好图片

个人爱好图片已经配置在 `src/components/about/about-content.tsx` 中：

```tsx
const hobbies = [
  {
    name: "咖啡",
    image: "/images/hobbies/coffee.jpg",
    // ...
  },
  // ...
];
```

## 🔄 添加新图片

### 方法 1：直接拖放

直接将图片文件拖放到对应的文件夹中即可。

### 方法 2：使用命令行

```bash
# 复制图片到封面目录
cp ~/Desktop/your-image.jpg public/images/posts/covers/

# 复制图片到内容目录
cp ~/Desktop/your-image.png public/images/posts/content/
```

### 方法 3：使用截图工具

使用系统截图工具保存到对应目录。

## 📏 图片建议尺寸

### 封面图

- **推荐尺寸**: 1200 x 630 px (16:9)
- **最小尺寸**: 800 x 450 px
- **格式**: JPG 或 PNG
- **大小**: 建议 < 500KB

### 内容配图

- **最大宽度**: 1200 px
- **格式**: PNG（带透明背景）或 JPG（照片）
- **大小**: 建议 < 300KB

### 爱好图片

- **推荐尺寸**: 800 x 600 px (4:3)
- **格式**: JPG 或 PNG
- **大小**: 建议 < 200KB

## 🎨 图片优化建议

1. **压缩图片**: 使用 TinyPNG (https://tinypng.com) 或 ImageOptim 压缩图片
2. **选择合适格式**:
   - 照片使用 JPG
   - 带透明背景使用 PNG
   - 图标和插图考虑使用 WebP
3. **控制尺寸**: 不要上传超过 2MB 的单张图片

## 📦 Git 提交建议

图片文件较大，建议：

1. 将图片添加到 `.gitignore`（如果是自动生成的图）
2. 或者使用 Git LFS (Large File Storage)
3. 定期清理不需要的图片

```bash
# 查看图片占用空间
du -sh public/images/

# 查找大文件
find public/images/ -type f -size +500k
```

## 📝 文档中的图片

`docs/goodblog/` 文件夹中的文档文件可能包含第三方图片链接。这些是参考文档，不影响博客前台显示。如需替换：

1. 下载文档中的图片到 `public/images/docs/` 目录
2. 将 markdown 中的图片链接从 `https://...` 改为 `/images/docs/filename.jpg`

**示例**:

```markdown
# 之前

![示例图片](https://youke1.picui.cn/s1/2025/10/21/68f732b6a9b93.png)

# 之后

![示例图片](/images/docs/example.png)
```
