## 📝 文档编写规范

### 新增文档

1. **选择合适的位置** - 根据文档类型选择对应的目录
2. **使用清晰的标题** - 便于目录索引
3. **添加代码示例** - 提供可运行的示例代码
4. **包含注意事项** - 标注重要的注意事项和常见问题
